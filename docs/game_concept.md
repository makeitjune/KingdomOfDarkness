# Game Concept

## Working Title

KingdomOfDarkness

## Genre

2D quarter-view RPG.

## Reference Feeling

- 어둠의 전설
- 아스가르드
- Old Korean online RPG field hunting feeling

## Core Fantasy

The player is not alone.

The player travels with AI companions who:

- Follow naturally.
- Join fights.
- React to danger.
- Speak in small situational lines.
- Grow with the player.
- Eventually feel like a party.

## MVP Fantasy

The first prototype should make the player feel:

```text
"내가 움직이면 동료가 따라오고,
몬스터를 치면 동료가 같이 싸워준다."
```

## Main Pillars

### 1. Quarter-view Movement

Movement must feel like old 2D quarter-view RPG movement.

### 2. Companion Presence

Companion should not be just a stat bonus.
The companion should visibly move, fight, and react.

### 3. Simple Hunting Loop

```text
Move → encounter → fight → kill → gain EXP → grow
```

### 4. Expandable RPG Systems

Later:

- Skills.
- Equipment.
- Quests.
- Towns.
- Multiple companions.
- More personality.

But MVP must stay small.

## Visual Style

Initial:

- Placeholder diamond tiles.
- Simple colored character placeholders.
- Basic health bars.
- Speech bubbles.

Later:

- Pixel-art or pre-rendered 2D sprites.
- Quarter-view environment tiles.
- Directional animations.

## Tone

Dark fantasy but not necessarily horror.

Working mood:

```text
lonely world + loyal companions + dangerous hunting fields
```
